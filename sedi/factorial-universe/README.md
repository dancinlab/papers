# The Factorial Universe

**How 3! Generates Physics, Computes Reality, and Bootstraps Consciousness — Evidence for the Fifth Revolution**

## Quick Start

```bash
# Setup environment
conda env create -f environment.yml
conda activate factorial-universe

# One-click reproduction
bash run_all.sh
```

## Structure

| File | Purpose |
|------|---------|
| `paper.tex` | LaTeX source |
| `notebooks/01-06` | Reproducible analyses |
| `data/` | Input data + download script |
| `figures/` | Auto-generated by notebooks |
| `sedi/` | Symlink to SEDI analysis engine |

## Data Requirements

- Breakthrough Listen Voyager filterbank (504 MB, in `~/Dev/sedi/data/seti/`)
- ANU QRNG API (live, no key needed)
- NASA Exoplanet Archive TAP (live, no key needed)
